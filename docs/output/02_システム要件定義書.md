## 3. 各コンポーネント詳細

### C4: AppSheet — 要改修（データソース変更）

#### 変更前
| 項目 | 内容 |
|---|---|
| メインテーブル | Form Responses 2（アンケート回答のみ） |

#### 変更後
| 項目 | 内容 |
|---|---|
| メインテーブル | 顧客マスタ統合シート（ANDPADの全顧客） |
| サブテーブル | 編集データシート（AppSheet加筆データ、SystemIDでRef） |
| 参照マスタ | ユーザーマスタ、スタジオマスタ |
| Form Responses 2 | AppSheetからは非表示 |

#### Security Filter（クロススタジオ対応 — 顧客単位 D13）

顧客マスタ統合シートのSecurity Filter:
```
OR(
  LOOKUP(USEREMAIL(), "ユーザーマスタ", "メールアドレス", "役割") = "管理者",

  AND(
    LOOKUP(USEREMAIL(), "ユーザーマスタ", "メールアドレス", "役割") = "エリアマネジャー",
    IN([担当者所属店舗],
      SELECT(スタジオマスタ[スタジオ名],
        [エリア] = LOOKUP(USEREMAIL(), "ユーザーマスタ", "メールアドレス", "エリア")))
  ),

  [担当者所属店舗] = LOOKUP(USEREMAIL(), "ユーザーマスタ", "メールアドレス", "スタジオ名"),

  IN(LOOKUP(USEREMAIL(), "ユーザーマスタ", "メールアドレス", "スタジオ名"),
    [編集データ].[閲覧許可スタジオ])
)
```

> 最後の条件: 編集データシートの「閲覧許可スタジオ」にスタッフのスタジオ名が含まれている場合、
> その顧客はスタッフに表示される。管理者が顧客単位で設定する。

#### AppSheet Action: 閲覧スタジオを設定
| 項目 | 設定 |
|---|---|
| Action名 | 閲覧スタジオを設定 |
| 対象テーブル | 編集データシート |
| Do this | App: go to another view within this app |
| Target | LINKTOROW([_THISROW], "閲覧スタジオ設定")（閲覧許可スタジオ列のみ表示するformView） |
| 表示条件 | `OR(LOOKUP(USEREMAIL(),"ユーザーマスタ","メールアドレス","役割")="管理者", LOOKUP(USEREMAIL(),"ユーザーマスタ","メールアドレス","役割")="エリアマネジャー")` |
| Position | Prominent |

> **スタッフにはこのボタンが表示されない**。管理者・エリアマネジャーが顧客詳細画面から
> 「閲覧スタジオを設定」→ スタジオをチェックボックスで選択 → 選択されたスタジオのスタッフもこの顧客を閲覧可能になる。

### C8: ANDPADインポート用xlsx生成 — 要改修
| 項目 | 変更前 | 変更後 |
|---|---|---|
| データソース | Form Responses 2 | **編集データシート** |
| 抽出条件 | 完了フラグ=TRUE AND 未連携 | 同じ |
| 更新キー | — | SystemID（既存顧客の更新用） |

### C10: アンケート→編集データ転記（新規開発）
| 項目 | 内容 |
|---|---|
| トリガー | Form Responses 2への書き込み時 or 日次バッチ |
| 処理 | アンケート回答データを編集データシートにコピー（SystemID付き） |
| 目的 | アンケートデータもAppSheetの編集データと同じ経路でANDPADに連携 |

## 5. 非機能要件（システム観点）

### 5.1 パフォーマンス
| 項目 | 要件 |
|---|---|
| WebApp画面表示 | 3秒以内 |
| 郵便番号→住所 | 2秒以内 |
| 既存顧客検索 | 5,000件から3秒以内（スタジオフィルタ済み） |
| AppSheet表示 | 顧客マスタ5,000件の中からスタジオ別にフィルタ表示 |
| xlsx生成 | 100件で30秒以内 |

### 5.2 セキュリティ
| 項目 | 要件 |
|---|---|
| WebAppアクセス | `access=DOMAIN`（組織内のみ） |
| プリフィル | 氏名・フリガナ・電話番号のみ（住所・メールは除外） |
| AppSheetアクセス | Security Filterでスタジオ別アクセス制御 |
| クロススタジオ | ユーザーマスタの「追加閲覧スタジオ」で制御 |
