## 5. 合格基準

| レベル | 基準 |
|---|---|
| UT | 各関数が期待する入出力を返す |
| IT | コンポーネント間でデータが正しく受け渡される |
| E2E | アンケート→ANDPAD登録まで人手介入なしで完了（新規・既存の両パターン） |
| UAT | パイロットスタジオのスタッフが「手書きより楽」かつ「全顧客が見えて便利」と判断 |
